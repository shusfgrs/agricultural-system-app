# 智慧农业控制系统

基于 OpenHarmony/HarmonyOS 的智慧农业管理 APP，通过蓝牙 BLE 与 ESP32 硬件通信，实现农作物环境的远程监测与智能控制。

## 项目概述

本项目是一个软硬件结合的智慧农业解决方案，包含：
- **HarmonyOS APP**：运行在手机/平板上的控制应用
- **ESP32 固件**：运行在硬件设备上的传感器数据采集与控制程序

### 主要功能

| 功能模块 | 描述 |
|----------|------|
| 智能控制中心 | 环境数据实时展示、蓝牙连接状态、模拟数据提示 |
| 智能灌溉 | 土壤湿度监测、自动灌溉、灌溉水量计算 |
| 智能光照 | 光照强度监测、LED补光控制、定时开关 |
| 阈值预警 | 多参数阈值配置、异常状态预警、预警历史记录 |
| 设备管理 | BLE设备扫描、连接管理、设备状态显示 |

### 监测参数

- 温度 (°C)
- 湿度 (%)
- 土壤湿度 (%)
- 光照强度 (lux)
- 水位 (%)
- CO2浓度 (ppm)

## 快速开始

### 环境要求

| 工具 | 版本 | 用途 |
|------|------|------|
| DevEco Studio | 4.0+ | HarmonyOS 应用开发 |
| Arduino IDE | 2.0+ | ESP32 固件开发 |
| Node.js | 16+ | 构建工具依赖 |

### APP 开发

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd MyApplication3
   ```

2. **打开项目**
   - 启动 DevEco Studio
   - 选择 "Open Project"
   - 选择项目根目录

3. **运行应用**
   - 连接 HarmonyOS 设备 或 启动模拟器
   - 点击 Run 按钮

4. **预览模式**
   - 打开任意 `.ets` 文件
   - 点击右侧 Previewer 面板

### 硬件开发

1. **安装 Arduino IDE**
   - 下载：https://www.arduino.cc/en/software

2. **配置 ESP32 开发环境**
   - 添加开发板管理器 URL：
     ```
     https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
     ```
   - 安装 ESP32 开发板支持

3. **安装依赖库**
   - DHT sensor library
   - ArduinoJson

4. **上传固件**
   - 打开 `hardware/esp32_smart_agriculture/esp32_smart_agriculture.ino`
   - 选择开发板：ESP32 Dev Module
   - 选择端口
   - 点击上传

详细步骤请参考 [硬件接线说明](hardware/esp32_smart_agriculture/README.md)

## 数据通信协议

### 传感器数据格式（硬件 → APP）

```json
{
  "temperature": 25.5,
  "humidity": 65,
  "soilMoisture": 45,
  "lightLevel": 300,
  "waterLevel": 75,
  "co2": 450,
  "timestamp": 1700000000
}
```

### 控制命令格式（APP → 硬件）

```json
{
  "cmd": "irrigation",
  "value": 1,
  "timestamp": 1700000000000
}
```

**支持的命令：**

| cmd | value | 说明 |
|-----|-------|------|
| irrigation | 0/1 | 灌溉开关 |
| light | 0/1 | 灯光开关 |
| brightness | 0-100 | 灯光亮度 |
| target_moisture | 20-80 | 目标湿度 |

详细协议请参考 [硬件组接口文档](docs/HARDWARE_INTERFACE.md)

## 功能说明

### 1. 智能控制中心（首页）

- 显示蓝牙连接状态
- 未连接时提示模拟数据模式
- 展示所有环境参数

- 手动/自动补光模式
- LED亮度调节（0-100%）
- 定时开关设置
- 实时光照强度监测

### 4. 阈值预警

- 6个参数的阈值配置
- 实时预警状态显示
- 预警历史记录

### 5. 设备管理

- BLE设备扫描
- 设备连接/断开
- 连接状态显示

## 注意事项

1. **模拟数据模式**：未连接蓝牙设备时，APP 使用模拟数据进行演示
2. **蓝牙权限**：真机运行需要授予蓝牙权限
3. **硬件预热**：CO2传感器（MQ-135）需要预热 24-48 小时
4. **供电要求**：继电器需要 5V 供电，传感器可使用 3.3V
